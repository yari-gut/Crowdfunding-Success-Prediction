# Datasets

## [Kickstarter Projects](https://www.kaggle.com/datasets/ulrikthygepedersen/kickstarter-projects?resource=download):

The Kickstarter dataset contains information about various crowdfunding projects hosted on the Kickstarter platform. It includes features such as project name, category, funding goal, pledged amount, number of backers, and project status (successful or failed). This dataset is used for training and evaluating machine learning models to predict the success of crowdfunding campaigns.

## [Indiegogo Crowdfunding Campaigns](https://www.kaggle.com/datasets/quentinmcteer/indiegogo-crowdfunding-data):

The Indiegogo dataset contains information about crowdfunding projects hosted on the Indiegogo platform. Similar to the Kickstarter dataset, it includes features such as project name, category, funding goal, pledged amount, number of backers, and project status. This dataset is also utilized for training and evaluating machine learning models to predict campaign success.
